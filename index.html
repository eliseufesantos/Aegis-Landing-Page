O Problema

A poluição nos oceanos tem sido um grande problema mundial. Resíduos plásticos, esgoto e poluentes industriais contaminam nossas águas, ameaçando a vida marinha, ecossistemas costeiros e a saúde humana. Somado a isso, faltam dados de qualidade para os algoritmos de centros de pesquisa, além da pouca conscientização da população sobre os problemas hídricos.

Tecnologias

  • 1

    Linguagens de Programação

    Python e JavaScript. Python para análise de dados e machine learning, enquanto JavaScript (com React) para criar uma interface interativa.

  • 2

    Frameworks

    Django para o backend; TensorFlow e Scikit-learn para machine learning e análise preditiva; e Leaflet.js para mapas interativos.

  • 3

    Geolocalização

    APIs como Google Maps e OpenStreetMap para capturar e exibir a localização geográfica dos dados coletados.

  • 4

    Notificações em Tempo Real

    Firebase Cloud Messaging para enviar alertas aos usuários sobre mudanças significativas na qualidade da água.

  • 5

    Banco de Dados

    PostgreSQL, pela sua capacidade de lidar com grandes volumes de dados geoespaciais.

  • 6

    Integração com APIs

    APIs de organizações ambientais e institutos de pesquisa para ampliar o uso e a validade dos dados coletados.

A Solução

A Aegis desenvolverá um aplicativo que reúne informações sobre praias, rios, cachoeiras e fontes de água, inicialmente coletadas da COI. Os usuários poderão adicionar dados locais, criando uma comunidade engajada em conscientização ambiental. As informações serão validadas por autoridades e centros de pesquisa, alimentando algoritmos e promovendo a importância da saúde dos ecossistemas aquáticos.

Objetivos

  • #

    Monitoramento de dados

    Estimular cidadãos a contribuir com dados valiosos sobre a qualidade da água. Estabelecer uma base de dados colaborativa e precisa para monitorar a saúde dos oceanos, rios e lagos

  • #

    Aumentar a Conscientização

    Fornecer informações precisas para apoiar políticas públicas e ações de conservação. Promover a conscientização sobre a importância da saúde dos corpos d'água entre a população.

  • #

    Educação Ambiental

    Oferecer recursos educativos e tutoriais sobre conservação e sustentabilidade. Engajar a comunidade em atividades de preservação e práticas sustentáveis.

O Público

A Aegis é destinada a cidadãos preocupados com o meio ambiente, pesquisadores, instituições educacionais, ONGs e órgãos governamentais. Envolvendo desde moradores locais que desejam monitorar suas praias e rios até cientistas que necessitam de dados precisos para suas pesquisas, nossa plataforma visa engajar uma comunidade comprometida com a sustentabilidade.
